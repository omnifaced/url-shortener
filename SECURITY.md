# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates:

| Version | Supported |
| ------- | --------- |
| 1.x.x   | ✅ Yes    |

---

### Reporting a Vulnerability

Please report security issues via GitHub Security Advisories:
Security → Advisories → Report a vulnerability

---

## Disclosure Process

1. The report will be acknowledged within **72 hours**
2. The vulnerability will be investigated and validated
3. A fix will be developed and released

---

## Security Updates

Security fixes are released as **patch versions** (e.g. `v1.0.1`) and will not introduce breaking changes.

---

Thank you for helping keep this project secure ❤️
